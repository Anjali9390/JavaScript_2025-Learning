Description

<b> → Makes text bold

<i> → Italic style text

<mark> → Highlights text

<small> → Makes text smaller

<del> → Shows deleted text

<ins> → Shows inserted text

<sub> → Subscript (used in formulas like H₂O)

<sup> → Superscript (used in formulas like x²)
